# 📝 Todo List App

A simple and responsive Todo List web application built using **HTML**, **CSS**, and **JavaScript**. This application helps users organize their daily tasks by allowing them to add, complete, and delete tasks easily.

## ⚙️ Installation & Usage

1. Clone this repository

```bash
git clone https://github.com/icebearrr99/todo-list-app.git
```

2. Open the project folder

```bash
cd todo-list-app
```

3. Run the application

Simply open `index.html` in your browser.
